# quinielaJoanSantos — diseño profesional

## Publicar actualización
1. Descomprime el ZIP.
2. En GitHub abre tu repositorio.
3. Sube `app.py`, `requirements.txt`, `.streamlit/` y toda la carpeta `assets/`.
4. Confirma con **Commit changes**. Streamlit se actualizará automáticamente.

## Mejoras incluidas
- Alto contraste y textos legibles.
- Diseño responsivo para celular.
- Tarjetas de partido con escudos y marcador centrado.
- Escudo visible al elegir participante, Survivor y resultados de administrador.
- Logo propio de quinielaJoanSantos.

Los escudos incluidos son identificadores visuales personalizados para la quiniela y pueden sustituirse por archivos oficiales manteniendo los mismos nombres.
